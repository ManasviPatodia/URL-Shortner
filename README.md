# URL Shortener

A full-stack URL Shortener application that allows users to generate short, shareable links for long URLs. It provides secure authentication, URL management, and click tracking through a simple and responsive interface.

## Features

- User authentication
- Shorten long URLs
- Custom short links
- URL redirection
- Dashboard to manage links
- Click analytics
- Responsive UI

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
